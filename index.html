📰Voice Newspaper

Read News in Your Language - Powered by AI Text-to-Speech

About This Project

Voice Newspaper is an innovative Python-based application designed specifically for visually impaired users. It extracts news content from websites and automatically converts it into natural-sounding audio using advanced Text-to-Speech technology.

Purpose: To make news and information accessible to everyone, regardless of visual ability.

Key Features

📰 Web Scraping

Automatically extract articles from news websites with intelligent content filtering

🔊 Text-to-Speech

High-quality audio conversion using Google's TTS technology

🌍 Multi-Language

Full support for English, Hindi, Tamil, and Malayalam

📄 PDF Support

Convert PDF documents to audio format

🎤 Speech Recognition

Convert audio files back to text

🎨 User-Friendly GUI

Easy-to-use Tkinter interface for all users

Quick Start Guide

Requirements:

  • Python 3.7 or higher
  • pip (Python package manager)
  • Windows, Linux, or macOS
  • Internet connection

Step 1: Clone the Repository

git clone https://github.com/Gautam-kumar12-creater/Voice-Newspaper-Web-Scraping.git
cd Voice-Newspaper-Web-Scraping

Step 2: Install Dependencies

pip install -r requirements.txt

Step 3: Run Your Preferred Language Version

# English
cd Audio_English && python gui.py

# Hindi
cd Audio_Hindi && python gui.py

# Tamil
cd Audio_Tamil && python gui.py

# Malayalam
cd Audio_Malayalam && python gui.py

How to Use

  1. Launch the Application - Run the gui.py file for your preferred language
  2. Choose Input Mode - Select "URL" (news website) or "Text" (manual input)
  3. Enter URL or Text - Paste a news link or type your text
  4. Click "View Result" - Extract and process the content
  5. Listen to Audio - Audio file plays automatically in your browser
  6. Enjoy! - Read along or listen hands-free
