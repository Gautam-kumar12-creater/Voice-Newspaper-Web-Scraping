🔧 Installation & Setup

Requirements:

  • Python 3.7 or higher
  • Internet connection (for news extraction & audio)
  • Modern web browser (Chrome, Firefox, Safari, Edge)

Run Locally:

cd Voice-Newspaper-Web-Scraping
python -m pip install -r requirements.txt
python app.py

Then open: http://localhost:5000

🌐 Supported News Websites

Extract news from popular Malayalam sources:

  • Manorama Online (manoramaonline.com)
  • Mathrubhumi (mathrubhumi.com)
  • Onmanorama (onmanorama.com)
  • AsiaNet News (asianetnews.tv)
  • Any other Malayalam news website

ℹ️ About This Project

Voice Newspaper is designed to make news accessible to visually impaired users by converting text articles into natural-sounding audio.
